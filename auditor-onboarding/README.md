# Onboarding "Skill Development" Tasks
For the AIEA Auditor program (CMPM 118) you will do 10 onboarding
tasks over the 10 week quarter.  (Note: the first task and the last
task are the same for each project.

After task 1, you will choose your project, which have tasks 2-9
defined in the respective folders.

To work on these tasks, you can clone the full repository.
